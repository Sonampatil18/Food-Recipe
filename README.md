# 🍽️ Food Recipe Corner

A responsive Food Recipe Web Application built using **HTML, CSS, and JavaScript** that allows users to search for recipes and view detailed recipe information using the Spoonacular API.

---

## 🚀 Features

* Search recipes by name
* Fetch recipe data from Spoonacular API
* Display recipe images and titles
* View detailed recipe information
* Responsive and user-friendly interface
* Fast and simple recipe browsing

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Spoonacular API

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/food-recipe-corner.git
```

### 2. Open the Project Folder

```bash
cd food-recipe-corner
```

### 3. Get a Spoonacular API Key

Visit:

https://spoonacular.com/food-api

Create an account and generate your API key.

### 4. Configure API Key

Open `script.js` and replace:

```javascript
let apikey = "YOUR_API_KEY";
```

with:

```javascript
let apikey = "YOUR_ACTUAL_API_KEY";
```

### 5. Run the Project

Simply open `index.html` in your browser.

---

## 💡 How It Works

1. Enter a recipe name in the search box.
2. Click the Search button.
3. The application fetches recipes from the Spoonacular API.
4. Matching recipes are displayed with images and titles.
5. Click "View Recipe" to see detailed recipe information.
